# Как создать KV Namespace в Cloudflare Dashboard

## Пошаговая инструкция:

1. **В Dashboard:**
   - Вы уже в разделе **Workers & Pages**
   - Вверху страницы найдите вкладки: **Overview**, **KV**, **R2**, **D1** и т.д.
   - Нажмите на вкладку **"KV"**

2. **Создайте namespace:**
   - Нажмите кнопку **"Create a namespace"** (обычно справа вверху)
   - Введите название: `GAMES_KV`
   - Нажмите **"Add"** или **"Create"**

3. **Скопируйте ID:**
   - После создания в списке namespace найдите `GAMES_KV`
   - Скопируйте **ID** (длинная строка, например: `abc123def456ghi789`)

4. **Вставьте в wrangler.toml:**
   - Откройте файл `wrangler.toml`
   - Найдите строку 19: `id = "your-kv-namespace-id"`
   - Замените на: `id = "ваш-скопированный-id"`

5. **Задеплойте:**
   ```bash
   git add wrangler.toml
   git commit -m "Add KV namespace ID"
   git push
   ```

## Альтернативный путь (если не видите вкладку KV):

1. Откройте ваш worker **"123"**
2. Перейдите в **Settings**
3. Найдите секцию **"KV Namespace Bindings"**
4. Нажмите **"Add binding"**
5. Там можно создать новый namespace или привязать существующий

## После настройки:

- Все созданные игры будут сохраняться в KV
- Данные будут доступны всем пользователям
- Страницы игр будут работать по ссылкам `/app/{id}/{name}/`
